## 🏗️ Tech Stack (Common across projects)

- **Backend**: Java 21, Spring Boot 3.2+, Spring Web, Spring Data JPA, Hibernate
- **Database**: MySQL / PostgreSQL
- **Security**: Spring Security + JWT + Role-based Authorization
- **Mapping**: MapStruct (clean DTO ↔ Entity conversion)
- **Validation**: Jakarta Bean Validation
- **Tools**: Lombok, Spring Boot DevTools, Maven
- **Frontend** (future): React / Next.js + Tailwind / Shadcn UI
- **Deployment** : Render, Railway, Vercel

## 📌 Project Goals & Learning Focus

- Master **clean layered architecture** (Controller → Service → Repository)

- Real-world **security implementation** (JWT, refresh tokens, roles)
- Transaction management & ACID properties
- Global exception handling & meaningful API responses
- Documentation & testing mindset
- Learn in public → share mistakes, learnings & code regularly

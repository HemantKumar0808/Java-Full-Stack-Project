# Bank-Management-System

This project is a Full Stack  Bank Management System implemented in JAVA. It allows users to create accounts, deposit and withdraw money, and view account details. The system uses file handling to store account information persistently.
## Features
- Create new bank accounts with unique account numbers.
- Deposit and withdraw money from accounts.
- View account details, including balance and transaction history.
- Persistent storage of account information using file handling.
## Technologies Used
- Java: The programming language used for the implementation of the system.
- File Handling: Used for storing account information persistently.
## How to Run
1. Clone the repository to your local machine.
2. Compile the Java files using a Java compiler.
3. Run the main class to start the application.
4. Follow the on-screen prompts to interact with the system.
